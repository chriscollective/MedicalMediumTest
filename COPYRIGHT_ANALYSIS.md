## 📚 1. 版權法基本概念

### 1.1 受保護的內容

**✅ 著作權保護的內容**：
- 書籍的**原創文字表達**
- 書籍的**編排結構**
- 書籍的**插圖、圖表**
- 書籍的**整體設計**

**❌ 不受保護的內容**：
- **事實陳述**（例如：「西芹含鈉」）
- **通用知識**（例如：「肝臟有解毒功能」）
- **概念和想法**（只保護表達方式，不保護想法本身）
- **已過期的版權**（通常是作者死後 50-70 年）

### 1.2 台灣著作權法相關條文

#### 📖 第 52 條：引用
> 為報導、評論、教學、研究或其他正當目的之必要，在合理範圍內，得引用已公開發表之著作。

**關鍵要件**：
1. **正當目的**：教學、研究等
2. **合理範圍**：不能大量抄襲
3. **已公開發表**：出版的書籍符合
4. **需註明出處**：必須標示來源

#### 📖 第 65 條：合理使用
判斷是否為合理使用的四大要素：
1. **利用目的及性質**（商業 vs 非商業、教育性）
2. **著作性質**（事實性 vs 創意性）
3. **利用質量及比例**（引用多少內容）
4. **對原著市場影響**（會不會取代原書）

## 🚨 2. 你的專案的風險分析

### 2.1 目前狀況評估

#### ✅ 降低風險的因素

1. **教育性質**
   - 測驗是教學工具
   - 幫助讀者複習和理解
   - 可能符合「教學目的」

2. **非商業性**（假設）
   - 如果網站免費
   - 沒有廣告收益
   - 沒有收費機制

3. **引用比例可能合理**
   - 題目通常是改寫的知識點
   - 不是逐字複製整本書
   - 測驗形式不同於原書

4. **可能促進銷售**
   - 激發讀者興趣去買書
   - 不太可能取代原書
   - 可能是免費宣傳

#### ❌ 增加風險的因素

1. **未經授權**
   - 沒有取得出版社或作者許可
   - 沒有授權協議

2. **直接使用內容**
   - 如果題目直接複製書中文字
   - 如果引用原句作為選項
   - 如果解析直接抄書

3. **可能的商業性**
   - 有 Vercel Analytics（可能有流量變現計畫）
   - 未來可能加入廣告
   - 可能建立品牌價值

4. **涵蓋大量內容**
   - 20 題測驗可能涵蓋書籍核心內容
   - 多本書的完整題庫
   - 可能讓人不買書就能學到重點

### 2.2 具體風險等級

#### 🔴 高風險情況

**如果你的題目是這樣**：
```
題目：「根據《搶救肝臟》第 3 章，肝臟最重要的功能是什麼？」
選項 A：「肝臟是身體的化學工廠，負責超過 2000 種化學功能」（直接抄書）
選項 B：...
解析：「如書中第 45 頁所述：『肝臟每分每秒都在...』」（逐字引用）
```

**風險**：
- ❌ 大量逐字引用
- ❌ 明確標示章節頁碼（證明直接抄襲）
- ❌ 未改寫原文
- **可能被視為侵權**

#### 🟡 中風險情況

**如果你的題目是這樣**：
```
題目：「下列哪項是肝臟的主要功能？」
選項 A：解毒
選項 B：消化
選項 C：呼吸
選項 D：循環
解析：「根據醫療靈媒的說法，肝臟負責身體的解毒工作。」
```

**風險**：
- ⚠️ 改寫了概念
- ⚠️ 但仍明確引用「醫療靈媒」
- ⚠️ 灰色地帶
- **可能需要許可**

#### 🟢 低風險情況

**如果你的題目是這樣**：
```
題目：「肝臟的主要功能是？」
選項 A：解毒代謝
選項 B：消化食物
選項 C：血液循環
選項 D：氧氣交換
解析：「肝臟是人體重要的解毒器官。」
```

**風險**：
- ✅ 通用醫學知識
- ✅ 沒有直接引用特定書籍
- ✅ 事實陳述
- **風險較低**

## 💬 7. 常見問題

### Q1: 如果出版社要我下架怎麼辦？

**A**: 立即配合
1. 馬上下架網站
2. 發布道歉聲明
3. 刪除所有題庫
4. 保留對話記錄
5. 尋求和解方案

### Q2: 會不會被告？

**A**: 機率不高，但有可能
- 通常出版社會先發警告信
- 如果立即配合，很少走到法律程序
- 但不能完全排除

### Q3: 其他人的測驗網站都沒事？

**A**: 不代表你也沒事
- 很多侵權行為因為規模小未被注意
- 不代表合法，只是還沒被追究
- 建議主動降低風險

### Q4: 我有標示「非官方」就沒事了嗎？

**A**: 不一定
- 標示只能降低風險，不能免除責任
- 還是要看具體使用內容
- 建議加上更完整的免責聲明

1. ✅ 取得授權
